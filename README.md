# angularjs2 下拉控件demo

基于http://valor-software.com/ng2-bootstrap/#/dropdowns
做的一个下拉列表控件
* 列表内容由父组件传递
* 子组件选择框发生变化后会通知父组件

# 运行效果
![image](https://github.com/ParadeTo/ng2-dropdown-demo/blob/master/imgs/demo.gif)

# 开始
```
git clone https://github.com/ParadeTo/ng2-dropdown-demo.git
npm install
npm start
```
